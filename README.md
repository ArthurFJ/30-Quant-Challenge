# 30-Quant-Challenge

Overview
--------

This repository contains a 30-day challenge focused on quantitative analysis, modeling, and data-driven methods. Each day includes a small project with reproducible code and documentation.

Objectives
---------

- Learn quantitative techniques and practical implementations.
- Build reproducible notebooks and scripts for analysis and modeling.

Audience
-------

Students and practitioners with Python and math background seeking structured daily exercises.

Structure & Flow
---------------

Top-level layout mirrors the other challenge repos and contains Templates, Days, and supporting files.

How to use
---------

1. Create a virtual environment.
2. Install dependencies: `pip install -r requirements.txt`.
3. Open the desired day's notebook and run.
